# lalg

Python library to experiment with Linear Algebra concepts

`lalg` stands for "**L**inear **Alg**ebra."

## Prerequisites

- [Python 3](https://www.python.org/)
- [numpy](http://www.numpy.org/)

## Installation

### Interactively

From the terminal

```sh
$ git clone https://github.com/erictleung/lalg.git
$ cd lalg
$ python3
```
